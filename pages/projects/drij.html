DRIJ Test Engine

Propulsion Rapid Development CAD
2023

Overview

The DRIJ is a rapid development test bed for liquid propulsion research. This project focuses on creating a low-cost, iterative platform for testing various propulsion concepts, currently utilizing N2O and Ethanol as propellants.

Technical Details

  • Propellants: N2O (Oxidizer) and Ethanol (Fuel)
  • Development: Mk 1 version currently in progress
  • Design Philosophy: Rapid iteration and testing
  • Objectives: Low-cost development platform

Development Process

The Mk 1 engine represents the first iteration of this test platform, focusing on basic functionality and reliable operation. Future iterations will incorporate lessons learned and potentially explore more advanced features based on test results.
